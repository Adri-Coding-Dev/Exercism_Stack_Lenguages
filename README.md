# Exercism Solutions Repository

Este repositorio contiene **soluciones comentadas de varios tracks de Exercism**, incluyendo Java, Python, C y otros lenguajes. El objetivo es **aprender, practicar y tener referencia de buenas prácticas** en distintos lenguajes de programación mediante ejercicios resueltos y explicados paso a paso.

---

## 🏁 Propósito

El propósito de este repositorio es:  
- Documentar soluciones claras y comentadas para ejercicios de Exercism.  
- Facilitar el aprendizaje de conceptos de programación y patrones de diseño en diferentes lenguajes.  
- Servir como referencia para estudiantes y desarrolladores que quieran mejorar sus habilidades.

## 📂 Estructura del repositorio

Las soluciones están organizadas por **lenguaje** y dentro de cada lenguaje por **ejercicio**:
```text
/solutions
│
├─ /Java
│ ├─ /Exercise1
│ │ └─ Solution.java
│ ├─ /Exercise2
│ │ └─ Solution.java
│ └─ ...
├─ /Python
│ ├─ /Exercise1
│ │ └─ solution.py
│ └─ ...
├─ /C
│ └─ ...
/ README.md
```

📝 Notas sobre estilo y comentarios

  Todas las soluciones incluyen comentarios explicativos sobre la lógica, decisiones y buenas prácticas.
